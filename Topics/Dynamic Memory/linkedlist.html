Linked Lists

Singly Linked List

A singly linked list is a data structure consisting of a sequence of nodes, where each node stores a value and a reference to the next node in the list. The first node is called the head and the last node is called the tail. Singly linked lists are useful for implementing stacks and queues and for solving other problems that involve sequential access to data.

Here is an example of how to create a singly linked list in C++: 

struct Node {
  int data;
  Node* next;
};

class LinkedList {
public:
  LinkedList() : head(nullptr) {}

  void push_front(int data) {
    Node* node = new Node{data, head};
    head = node;
  }

  void pop_front() {
    if (head) {
      Node* node = head;
      head = head->next;
      delete node;
    }
  }

private:
  Node* head;
};

In the above implementation, push_front adds a new node to the beginning of the list, and pop_front removes the first node from the list.

The time complexity for inserting or deleting a node at the front of a singly linked list is O(1). However, finding the nth node in the list or accessing a node in the middle of the list has a time complexity of O(n).

Doubly Linked List
